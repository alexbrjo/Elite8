Assembly is Not Scary

An interactive online book part of the Elite8 Intel 8008 simulator

Hey! yeah I code

Ok, you're a programmer that is familiar with a higher-level programming language (Java, c++, Python, JavaScript, Ruby etc), but doesn't have a good idea what happens at the byte code level. This mini-book will give you, a high-level programmer, a better idea of what happens when your code is compiled. (Interpreting is another layer of abstraction)

Um, I've never programmed before

Not sure how you got to this page, but go watch the visual simulations because they're pretty memorizing! Some of the demos do require a touch of knowledge of JavaScript, but the visualizations have a convenient play button so you can just sit back and watch.
