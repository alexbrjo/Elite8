What is a register?

physical variables in the microprocessor

A Register

A register is similar to what a higher-level programmer would call a local variable. Registers are basically local variables stored physically with multiple binary values in the processor.

In assembly you work very personally with the processor. The assembly grammar is taylored to the specific architecture and instruction set of the microprocessor.
